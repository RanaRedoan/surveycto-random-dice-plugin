# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-25

### Added

- Initial production-ready SurveyCTO dice plugin
- Single-die roll (`1..6`) with animated UI
- Lock-after-roll default behavior with optional re-roll
- Timestamp and RNG metadata logging
- Optional sound toggle (default off)
- Unit tests, CI workflow, examples, and project documentation

## [1.0.1] - 2026-02-25

### Fixed

- Rebuilt as a true SurveyCTO field plug-in package format
- Added `dice-roller.fieldplugin.zip` artifact for direct form attachment
- Updated sample XLSForm to use `custom-dice-roller` appearance
- Added ready form package ZIP containing both XLSForm and plug-in file
- Replaced non-numeric XLSForm version with numeric `2026022502`
