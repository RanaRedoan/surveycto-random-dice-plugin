# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-25

### Added

- Initial production-ready SurveyCTO dice plugin
- Single-die roll (`1..6`) with animated UI
- Lock-after-roll default behavior with optional re-roll
- Timestamp and RNG metadata logging
- Optional sound toggle (default off)
- Unit tests, CI workflow, examples, and project documentation
