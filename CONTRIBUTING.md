# Contributing

## Development setup

```bash
npm install
npm run check
```

## Commit style

Use Conventional Commits where possible:

- `feat: ...`
- `fix: ...`
- `docs: ...`
- `test: ...`
- `chore: ...`

## Pull requests

- Keep changes scoped and reviewable
- Add/update tests for behavior changes
- Update README when user-facing behavior/config changes
- Update CHANGELOG for releases
